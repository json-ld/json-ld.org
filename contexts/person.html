The Person context

The Person context is based on a combination of FOAF, schema.org , and vcard vocabularies. It defines the following terms:

foaf
http://xmlns.com/foaf/0.1/
schema
http://schema.org/
vcard
http://www.w3.org/2006/vcard/ns#
xsd
http://www.w3.org/2001/XMLSchema#
Address
vcard:Address
address
vcard:address
affiliation
schema:affiliation
born
schema:birthDate of type xsd:date
child
schema:children of type @id
colleague
schema:colleagues of type @id
country
vcard:country
depiction
foaf:depiction of type @id
died
schema:deathDate of type xsd:date
email
foaf:mbox of type @id
familyName
foaf:familyName
gender
schema:gender
givenName
foaf:givenName
homepage
foaf:homepage of type @id
honorificPrefix
schema:honorificPrefix
honorificSuffix
schema:honorificSuffix
image
foaf:img of type @id
jobTitle
foaf:title
knows
foaf:knows of type @id
locality
vcard:locality
name
foaf:name
nationality
schema:nationality
nickname
foaf:nick
parent
schema:parent of type @id
Person
foaf:Person
postalCode
vcard:postal-code
region
vcard:region
sibling
schema:sibling of type @id
spouse
schema:spouse of type @id
street
vcard:street-address
telephone
schema:telephone
