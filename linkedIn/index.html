JSON-LD Tool for LinkedIn


JSON-LD is a lightweight Linked Data format. The objective of this tool is tool is fetch JSON data from LinkedIn API and demonstrate the conversion of that into valid JSON-LD. All JSON-LD documents are a valid JSON too. So JSON-LD can be thought as an extension of JSON with @context. A @context is used define the short-hand names that are used throughout a JSON-LD document.


About this tool

This is a simple tool to generate JSON-LD document from LinkedIn data. At this current stage of the tool, after you sign-in using the Sign-in with LinkedIn button your data will be fetched from LinkedIn. linkedIn.js then converts this JSON to JSON-LD and displays it. You can also use the Advanced button to see the LinkedIn JSON data and @context. It will also allow to save the JSON-LD document and export feature that you could use it for personal purpose or publish on to your website.
